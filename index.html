FCPP - FieldCalc++

(website under construction)

FCPP is an efficient C++14 implementation of the Pure Field Calculus, for fast and effective simulation of pervasive computing scenarios, and deployment on microcontroller architectures.

The source code is available at: https://github.com/fcpp/fcpp.

Documentation of the API is available at: http://fcpp-doc.surge.sh.

A presentation of the tool is available at: http://giorgio.audrito.info/static/fcpp.pdf (will be presented at ACSOS 2020).

A sample FCPP project is available at: https://github.com/fcpp/sample-project. Works based on the FCPP library can be easily setup by starting from a clone of this project.
